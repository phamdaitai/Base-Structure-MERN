# BDS
## Server
**Cài đặt thư viện**
* Vào thư mục server, bật terminal và chạy lệnh > npm install

**Thêm file .env**

**Chạy chương trình**
* > npm run dev


## Client
**Cài đặt thư viện**
* Vào thư mục client, bật terminal và chạy lệnh > npm install

**Thêm file .env**

**Chạy chương trình**
* > npm start
* dev: http://localhost:3000
